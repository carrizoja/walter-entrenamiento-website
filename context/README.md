# Context Folder

This folder contains reference files, documentation, and resources that provide context for development.

## Purpose

Upload files here that AI agents and developers should reference when working on the project, such as:

- Design specifications
- API documentation
- Business requirements
- Data schemas
- Reference materials
- Sample data structures
- Project notes and decisions

## Usage

Files in this folder are meant to provide context and should not be imported directly into the application code. They serve as reference documentation for development.
